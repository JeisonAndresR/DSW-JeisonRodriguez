Proposito del sitio

Comprender la utilidad de las etiquetas semánticas y de las CSS Grid en las paginas web de la mano con el manejo de html y CSS

etiquetas semánticas

Las etiquetas semánticas nos permiten redefinir la forma de presentación de los navegadores web, ayudando a tener una estructura de vistas más homogeneizada, nos permite manejar un nuevo sistema de: separar totalmente la información y la forma de presentarla, resumir, simplificar y hacer la forma deber más sencilla. Diez T, Domínguez M, Martínez J, Sáenz J. 2012. Creación de Páginas WEB accesibles con HTML5 Recuperado de: http://www.esvial.org/wp-content/files/Atica2012_pp120-129.pdf

CSS Grid

Es un sistema de maquetación que da una hoja de estilos con una cuadricula a partir de la cual se puede posicionar los diferentes elementos de manera más sencilla, también es conocida como el método rejilla la cual se visualiza de manera que nos permite una exacta distribución de los elementos. Mariño M. agosto 2019. CSS Grid: Ventajas y primeros pasos. Recuperado de: https://dinahosting.com/blog/css-grid-ventajas-y-primeros-pasos/
